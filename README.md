**RESTFUL API**
```
POST—Create
GET—Retrieve
PUT / PATCH—Update
DELETE—Delete


Response Code	Description
200	Success
201	Created
204	No content
400	Bad request
403	Forbidden
404	Not found
500	Internal server error

```
